My works

Here is the work that I had the opportunity to do during my school trip.

Little Genius

A quizz game

Little Genius is a game developed in JAVA with the aim of knowing how to use the JAVA FX library. This game is based on the duel part of the Question pour un champion program.

Book A Zoo

For animals lovers

Book A Zoo is a website developed in order to interact with a database hosted on a server using PHP technology. You can consult information about the various European Zoo there and simulate a ticket purchase.

The Knight Rest

Adventure Game

The Knight Rest is a game developed in C ++ using the SMFL library. This project aims to be able to code easily in C ++ and to synthesize all the concepts seen during the course. In this game, you play as a knight who, despite the desire to take a good rest on a tropical island, finds himself forced to confront many enemies in turn-based combat.

Fitness Battle

Website and Android App for fitness lover

Fitness Battle is an Angular coded site allowing you to encode sports training, earn points and challenge your friends. You can also analyze your performance or take advantage of tools for fitness. An android application allows the same functionalities. Both projects use a Web API developed in C # communicating with a local database.
